## 截图规范

1. 分辨率要求
   - 界面截图：1920x1080 或更高
   - 架构图：建议使用矢量图格式（SVG/PDF）
   - 示例图片：根据实际需求调整

2. 命名规范
   - 使用小写字母
   - 单词间用连字符（-）分隔
   - 使用有意义的描述性名称

3. 格式要求
   - 界面截图：PNG 格式
   - 架构图：SVG/PDF 格式
   - 示例图片：PNG/JPG 格式

4. 文件大小
   - 单个文件不超过 2MB
   - 建议使用图片压缩工具优化
